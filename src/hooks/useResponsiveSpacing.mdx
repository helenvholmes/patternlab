# useResponsiveSpacing

| Hook Version | DS Version   |
| ------------ | ------------ |
| Added        | `Prerelease` |
| Latest       | `Prerelease` |

The `useResponsiveSpacing` hook returns responsive gap and padding values that
align with NYPL design standards for responsive grids and columns. The values
will adjust based on the width of the viewport.

The `Template` component uses the `useResponsiveSpacing` hook for margin and
grid spacing values.

## Usage

<Source
  code={`
import { 
  useResponsiveSpacing,
} from "@nypl/design-system-react-components";
// ...
const {
  responsiveGap,
  responsiveMargin,
  responsivePadding,
} = useResponsiveSpacing();
`}
  language="jsx"
/>

## responsiveGap

The `responsiveGap` object is intended to be used as the gap values for
repeating grids and table.

- Mobile: `1rem` (16px)
- Tablet: `1.5rem` (24px)
- Desktop: `2rem` (32px)

<Source
  code={`
const {
  responsiveGap
} = useResponsiveSpacing();
// ...
const gridStyles = {
  display: "grid",
  gap: responsiveGap,
  gridTemplateColumns: "repeat(3, 1fr)",
}
<div style={ gridStyles }>
  <div>...</div>
  <div>...</div>
  <div>...</div>
  <div>...</div>
  <div>...</div>
  <div>...</div>
</div>
`}
  language="jsx"
/>

## responsiveMargin and responsivePadding

The `responsiveMargin` and `responsivePadding` objects are intended to be used
as the values for the left and right spacing on any element that could
potentially touch the edge of the browser window. The values for each object are
the same, but separate objects have been provided to enable the use of the
object that aligns semantically with spacing methods used in consuming apps.

- Mobile: `1rem` (16px)
- Tablet: `1.5rem` (24px)
- Desktop: `1rem` (16px)

<Source
  code={`
const {
  responsivePadding,
} = useResponsiveSpacing();
// ...
const containerStyles = {
  maxWidth: "1280px",
  paddingLeft: responsivePadding,
  paddingRight: responsivePadding,
}
<div style={ containerStyles }>
  <div>...</div>
</div>
`}
  language="jsx"
/>

## Visual example

The example below illustrates the responsive padding and gap values in action.
The `padding` is orange and the `gap` is blue. Change the size of your browser
window and watch the spacing values adjust as the width of the window decreases
and increases.

export const { responsiveGap, responsiveMargin, responsivePadding } =
  useResponsiveSpacing();
export const containerStyles = {
  bgColor: "ui.warning.primary",
  maxWidth: "1280px",
  px: responsivePadding,
};
export const gridStyles = {
  bgColor: "ui.link.primary",
  display: "grid",
  gap: responsiveGap,
  gridTemplateColumns: {
    base: "1fr",
    sm: "repeat(2, 1fr)",
    md: "repeat(3, 1fr)",
    lg: "repeat(4, 1fr)",
  },
};
export const contentStyles = {
  bgColor: "ui.bg.default",
  color: "ui.typography.heading",
  p: "l",
  textAlign: "center",
};

<Box __css={containerStyles}>
  <Box __css={gridStyles}>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
    <Box __css={contentStyles}>Content</Box>
  </Box>
</Box>
