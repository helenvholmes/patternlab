## General Information

The Reservoir Design System (DS) adheres to the following breakpoints for
responsive layouts and typography.

- <strong>Small mobile:</strong> less than 480px
- <strong>Large mobile:</strong> less than 768px
- <strong>Small tablet:</strong> less than 1024px
- <strong>Large tablet:</strong> less than 1280px
- <strong>Desktop:</strong> greater than or equal to 1280px

## SCSS Variables

When using SCSS in an app:

- define an import for the DS `/dist/resources.scss` file
- use the following SCSS variables in media queries or SCSS custom `breakpoint` functions

<Source
  code={`
// Chakra syntax
$nypl-breakpoint-small: 480px;
$nypl-breakpoint-medium: 768px;
$nypl-breakpoint-large: 1024px;
$nypl-breakpoint-xl: 1280px;

// NYPL semantic syntax
$nypl-breakpoint-smallMobile: 480px;
$nypl-breakpoint-largeMobile: 768px;
$nypl-breakpoint-mobile: 768px;
$nypl-breakpoint-smallTablet: 1024px;
$nypl-breakpoint-tablet: 1280px;

$nypl-max-width: $nypl-breakpoint-xl;
`}
language="css"
/>
