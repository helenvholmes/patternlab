# Keyword search field

The `keyword search` field provides a way for users to enter a text string to
search and retrieve related information from a defined data set. It is not
considered a filter.

While the `keyword search` field is not required in a filtering scenario, more
often than not, filtering is used to narrow a results set that was initially
produced with a keyword search.

## Rendering

The `keyword search` field should be rendered using the DS
[SearchBar](./?path=/docs/components-form-elements-searchbar--docs) or
[TextInput](./?path=/docs/components-form-elements-textinput--docs) component.

## Position

The `keyword search` field should be positioned above the `filter bar` and `active filters` list.

## Anti-patterns

The `keyword search` field should not be considered when using [live
filtering](./?path=/docs/style-guide-filtering-applying-clearing-filters--docs#applying-filters).
Rather, a keyword search term should be applied using a submit button.
